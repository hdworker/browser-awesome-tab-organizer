# Yandex Tab Sorter

Расширение для умной сортировки (5 вариантов сортировки) и группировки вкладок браузера Яндекс.

Т.е. помимо сортировки вкладок,  приложение открывает столько окон браузера, сколько сайтов открыто у вас, вынося в отдельное окно каждого сайта вкладки сайта.

## Настройка локального репозитория

1. Установите Git на ваш компьютер, если он еще не установлен

2. Откройте терминал или командную строку

3. Перейдите в директорию, где хотите разместить проект

4. Клонируйте репозиторий одним из следующих способов:

   Для анонимного клонирования используйте SSH:

   ```
   git clone ssh://ssh.sourcecraft.dev/hdworker/yandex-tab-sorter.git
   ```

   Или используйте HTTPS с токеном доступа:

   ```
   git clone https://git@git.sourcecraft.dev/hdworker/yandex-tab-sorter.git
   ```

5. Перейдите в директорию проекта:

   ```
   cd yandex-tab-sorter
   ```

## Работа с репозиторием

### Выполнение коммитов

1. Добавьте измененные файлы в индекс:

   ```
   git add .
   ```

   или для конкретного файла:

   ```
   git add имя_файла
   ```

2. Создайте коммит с описанием изменений:

   ```
   git commit -m "Описание внесенных изменений"
   ```

### Получение изменений (Pull)

Для получения последних изменений с удаленного репозитория:

```
git pull origin main
```

### Отправка изменений (Push)

Для отправки ваших локальных коммитов в удаленный репозиторий:

```
git push origin main
```

Если вы работаете в другой ветке, замените "main" на имя вашей ветки.
