# Tab Sorter

Расширение для Яндекс Браузера, которое помогает быстро привести вкладки в порядок: отсортировать их, сгруппировать по доменам и сохранить рабочую сессию.

## Возможности

- сортировка по названию, домену, времени последнего доступа, favicon или протоколу;
- группировка вкладок по доменам в отдельные окна;
- сохранение вкладок в историю с возможностью закрыть исходные вкладки;
- избранное для важных вкладок;
- экспорт истории и избранного в HTML или CSV;
- светлая тема по умолчанию и переключатель тёмной темы.

## Установка для разработки

1. Клонируйте репозиторий:

   ```bash
   git clone https://github.com/hdworker/browser-awesome-tab-organizer.git
   cd browser-awesome-tab-organizer
   ```

2. Откройте страницу расширений Яндекс Браузера: `browser://extensions`.
3. Включите режим разработчика.
4. Выберите «Загрузить распакованное расширение» и укажите корень проекта.

## Использование

Откройте popup расширения через иконку на панели браузера. Выберите способ сортировки и нужное действие. Настройки и выбранная тема сохраняются синхронно в браузере.

## Структура

- `popup.html`, `popup.js` — интерфейс и действия popup;
- `history.html`, `history.js` — история и избранное;
- `background.js` — обработчик запуска сортировки через иконку расширения;
- `src/utils/database.js` — локальное хранилище IndexedDB.
